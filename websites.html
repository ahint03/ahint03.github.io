This collection showcases a series of web design projects focused on layout design, responsive interfaces, and interactive user experiences. Each project explores different approaches to structure, styling, and functionality while emphasizing usability and visual clarity.

I focused on creating a cohesive visual style through custom icons and UI elements while integrating live data to ensure accuracy and responsiveness.

Blackjack

Advanced Tic Tac Toe

An advanced version of tic tac toe consisting of nine smaller boards that form a larger game. Each move determines the next board the opposing player must play in, creating a layered strategic system. Winning a smaller board claims that space on the larger grid, with the overall goal of winning the meta-game. This project focused heavily on managing complex game logic and player flow.

Image Zoom

An interactive image tool that allows users to explore visual details dynamically. As the user moves their cursor across an image, a separate display updates to zoom into the corresponding area in real time. This project focused on user interaction, coordinate mapping, and creating a smooth and responsive visual experience.
